# Service Monitor Dashboard

A full-stack web app to monitor and visualize **backend service health** in real time.  
This dashboard displays **CPU usage, memory usage and service uptime** for servers or applications.

## Features

- Real-time monitoring of CPU, memory, and uptime metrics
- Interactive React dashboard built with Vite
- Node.js/Express backend providing a REST API
- Alerts for high CPU or memory usage
- Easy to extend for new metrics or additional monitoring

## Local Hosting

1. **Clone the repo**
- git clone https://github.com/perumallaharshitha/Service-Monitor-DashBoard.git
- cd Service-Monitor-DashBoard

2. **Start the Backend**
- cd backend
- npm install
- node server.js

3. **Start the Frontend**
- cd ../frontend
- npm install
- npm run dev

4. Open the LocalHost
